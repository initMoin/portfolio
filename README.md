# iOS 26 Micro-Apps Portfolio

A collection of small, surgical apps built with Swift 6, SwiftUI, and iOS 26 APIs.  
Each app demonstrates focused expertise with Apple’s modern frameworks.

## Apps
- **Pocket Pantry** – SwiftData CRUD + auto-migration (`@Model`, `@Query`).
- **Step Counter Widget** – WidgetKit + HealthKit + ActivityKit (in progress).
- **Hot-Cold Finder** – NearbyInteraction + MultipeerConnectivity.
- **Daily Quote Fetcher** – BackgroundTasks + UserNotifications.

## Structure
Each app is a standalone Xcode project inside its own folder.  
Repo root contains this README and shared docs.
